# Gamesub Scout: Game subscription database

## Intro

Gamesub Scout is a site that allows users to find what games are available on all the different video game subscription services, all in one place.

## Technologies and methodology

Gamesub Scout uses Javascrip, React, Vite, HTML, and CSS along with an API from PSPrices.com to compile a list of games availble on the major game subscription servcies.
